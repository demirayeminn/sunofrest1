# Sun of Rest

Antalya merkezli plaj ve otel ekipmanları firması için web sitesi.

## Kurulum

```bash
npm install
npm run dev
```

## Özellikler

- **Özgün tasarım**: Cormorant Garamond + Outfit font eşleşmesi
- **Akdeniz paleti**: Terracotta, deniz mavisi, kum tonları
- **Bölümler**: Hero, Ürün Kataloğu, Projeler, İletişim
- **Responsive**: Mobil uyumlu

## Katalog

"Katalog İndir" butonu `public/katalog.pdf` dosyasına yönlenir. PDF kataloğunuzu `public/katalog.pdf` olarak eklediğinizde buton çalışır.

## Sayfalar

- Ana Sayfa, Hakkımızda, Ürünler, Hizmet Alanları, Referanslar, İletişim (hepsi ayrı sayfa)

## Ürün Kategorileri

- Şezlong Modelleri
- Şezlong Sehpa Modelleri
- Masa Çeşitleri
- Sandalye Çeşitleri
- Duşluklar
- Havluluklar
